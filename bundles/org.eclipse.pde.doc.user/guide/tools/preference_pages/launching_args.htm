Launching Arguments

The Launching Arguments tab can be viewed as a template where you can specify program and VM arguments that you wish to test your plug-ins with.

Values entered on this tab are used to initialize the program and VM arguments on newly-created Eclipse Application, Equinox and JUnit Plug-in launch configurations.

For more information on running Eclipse and a list of program and VM arguments, refer to the following documents:

  • Running Eclipse
  • Eclipse Runtime Options
