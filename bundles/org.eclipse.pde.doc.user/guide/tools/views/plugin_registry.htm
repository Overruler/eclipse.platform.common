Plug-in Registry

The Plug-in Registry view shows the state of all plug-ins in the running instance of Eclipse. The view is available under Window > Show View > Other... > PDE Runtime > Plug-in Registry.

 

State of the Plug-in

For each plug-in in the running system, the view shows:

  • the plug-in ID and version
  • the plug-in active/inactive state, where the active state is indicated by a green overlay icon.
  • the plug-in location
  • the plug-in runtime libraries
  • the plug-in dependencies
  • the extensions and extension points the plug-in contributes

 

Plug-in Filtering

The Show Active Plug-ins menu item is a filter that removes all inactive plug-ins from the view. To re-add them, press the menu item again. This filtering action is also available from the view's toolbar drop down menu.
