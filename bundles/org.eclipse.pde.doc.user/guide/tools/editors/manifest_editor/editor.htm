Plug-in Manifest Editor

PDE provides a single form-based multi-page plug-in manifest editor that manages all plug-in files: MANIFEST.MF, plugin.xml/fragment.xml and build.properties. You use the plug-in manifest editor as though you're editing a single file, and PDE handles the task of writing the data to the different files on disk.

For details on the individual editor pages, refer to the following documents:

  • Overview
  • Dependencies
  • Runtime
  • Extensions
  • Extension Points
  • Build

 

Manifest Editor Presentation

By default, the plug-in manifest editor shows information and tabs about extensions, extension points and Equinox-specific MANIFEST.MF headers and attributes. If you do not wish to use these Eclipse-specific features, you can filter ou these pages from the editor on the Plug-in Development property page of your plug-in project via Properties > Plug-in Development from the context menu of plug-in projects.
