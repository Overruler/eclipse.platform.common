Included Features

The Included Features reference all features nested inside this feature and their properties.

 

Feature List

The feature list enumerates all the features that are directly nested and therefore considered part of this feature.

When the feature is built, all included features are recursively built and packaged.

 

Included Feature Details

An included feature may be marked as optional. Users may choose to not install optional features. They may also choose to disable them if they are installed or install them later. A missing optional feature is not treated as an error.

An included feature can be updated by patches. By default, Update Manager searches for updates to an included feature at the update site for the parent feature, but this setting can be overridden.

If an included feature contains platform-specific code or resources, then environment conditions should be specified and associated with that feature.
