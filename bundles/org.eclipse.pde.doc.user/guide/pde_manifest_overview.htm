Overview page

The Overview page shows you "plug-in at a glance" information grouped into sections. The most important information for each section is shown on the page, with hyperlinks and More buttons provided to access additional information.

The general plug-in information can be edited.  The rest of the information on this page can only be browsed.  If a plug-in class is specified for the plug-in, a hyperlink will open the class in a Java editor.

The Alerts section will bring to your attention problems and outstanding tasks associated with the plug-in. For example, it will flag compilation errors and warnings, as well as unresolved or cyclical references to other plug-ins.
